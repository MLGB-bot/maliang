三角形(triangle)
=====================

.. _triangle.py:      https://github.com/MLGB-bot/maliang/blob/main/examples/shapes/2D/triangle.py


::

    triangle(x1: float, y1: float, x2: float, y2: float, x3: float, y3: float)


* 参数 ::

        x1 第1个点x坐标
        y1 第1个点y坐标
        x2 第2个点x坐标
        y2 第2个点y坐标
        x3 第3个点x坐标
        y3 第3个点y坐标
